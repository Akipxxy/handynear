# HandyNear

HandyNear is a web platform that connects homeowners and businesses with trusted, verified handymen for repairs, installations, and maintenance work — covering plumbing, electrical, carpentry, painting, and general fixes.

## Features

- **Browse Handymen** — filter verified professionals by category (plumbing, electrical, carpentry, painting, general) and view ratings, location, and profile photos.
- **How It Works** — a simple overview of the process for customers.
- **Post a Job / Register** — a signup form for handymen to join the network.
- Responsive design with a mobile-friendly navigation menu.

## Tech Stack

- [React](https://react.dev/) 18
- [React Router](https://reactrouter.com/) for client-side routing
- Create React App (`react-scripts`) for build tooling
- Plain CSS (custom properties for theming, no framework)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or later recommended)
- npm (comes with Node.js)

### Installation

```bash
git clone https://github.com/Akipxxy/handynear.git
cd handynear
npm install
```

### Running locally

```bash
npm start
```

Opens the app at [http://localhost:3000](http://localhost:3000).

### Building for production

```bash
npm run build
```

Outputs an optimized production build to the `build/` folder.
